
# glm_color

A simple library for manipulating and generating color values.

This is an extension of [glm-rs](https:://www.github.com/dche/glm-rs).

# Usage

Add `glm_color` to the dependencies of `Cargo.toml`. For example,

```toml
[dependencies]
glm_color = "*"
```
